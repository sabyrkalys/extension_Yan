=============================================
 КАК ОБНОВИТЬ РАСШИРЕНИЕ ASTRA MAPS
=============================================

━━━ WINDOWS ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Запусти update_extension.bat
   (двойной клик, от обычного пользователя)
2. Скрипт сам скачает новые файлы с сервера
3. Открой chrome://extensions
4. Нажми 🔄 на расширении Astra Maps

━━━ ANDROID (Яндекс Браузер) ━━━━━━━━━━━━━
1. Открой браузер на планшете
2. Перейди по адресу:
   http://192.168.22.169:5001/version.json
   (проверь что видишь номер версии)
3. Скачай файлы по очереди:
   http://192.168.22.169:5001/content.js
   http://192.168.22.169:5001/background.js
   http://192.168.22.169:5001/inject.js
   http://192.168.22.169:5001/manifest.json
4. Замени файлы в папке расширения
5. Открой браузер → Расширения → 🔄

━━━ КАК ВЫПУСТИТЬ ОБНОВЛЕНИЕ (сервер) ━━━━
1. Обнови файлы в папке ws-server\extension\
2. В version.json увеличь номер версии:
   "version": "1.0.1"  (было 1.0.0)
3. Оповести расчёты — можно через WS-чат

Расчёты сами запускают update_extension.bat
и нажимают 🔄 — занимает 30 секунд.
