=============================================
 КАК УСТАНОВИТЬ SSL СЕРТИФИКАТ
=============================================

━━━ WINDOWS ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Скопируй trust_cert.bat в папку ws-server (рядом с cert.pem)
2. Правой кнопкой → "Запустить от имени администратора"
3. В появившемся окне нажми "ДА"
4. Перезапусти Chrome

Делается ОДИН РАЗ на каждом Windows-устройстве.

━━━ ANDROID (Chrome) ━━━━━━━━━━━━━━━━━━━━━
1. Переименуй cert.pem в cert.crt
   (или скачай готовый файл cert.crt)

2. Перешли файл cert.crt на планшет
   (через Telegram, почту, USB — любым способом)

3. Открой на планшете:
   Настройки → Безопасность → Ещё → Шифрование и учётные данные
   (на некоторых Android: Настройки → Безопасность → Установить сертификат)

4. Выбери "Сертификат CA" или "Сертификат центра сертификации"
   Найди файл cert.crt и нажми "Установить"

5. Введи PIN/пароль планшета если попросит

6. Перезапусти Chrome на планшете

ВАЖНО: На Android 11+ пользовательские сертификаты CA
не доверяются автоматически в Chrome.
Если шаги выше не помогли — используй метод ниже.

━━━ ANDROID — альтернативный метод ━━━━━━━
Если Chrome всё равно показывает предупреждение:

1. Открой Chrome на планшете
2. Перейди по адресу: https://192.168.22.103:5000
3. Нажми "Дополнительно" → "Перейти на сайт (небезопасно)"
4. Закрой вкладку

Chrome запомнит исключение для этого адреса.
Это нужно сделать ОДИН РАЗ на каждом планшете.
